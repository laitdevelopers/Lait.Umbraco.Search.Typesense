# Umbraco Typesense Search Integration

An Umbraco CMS (v15/16) integration for [Typesense](https://typesense.org/), modelled on the official
[Umbraco.Cms.Integrations.Search.Algolia](https://github.com/umbraco/Umbraco.Cms.Integrations/tree/main/src/Umbraco.Cms.Integrations.Search.Algolia)
package and adapted to the new (Lit/TypeScript) backoffice.

## Repository layout

```
src/Lait.Umbraco.Search.Typesense/
├── *.cs                     # Backend: composer, services, controllers, handlers, migrations…
├── Client/                  # Backoffice extension (TypeScript + Lit + Vite)
│   ├── src/                 # Dashboard element + API client
│   └── public/umbraco-package.json
└── App_Plugins/...          # Built client output (generated by `npm run build`)
```

## Build

Backend:

```bash
dotnet build src/Lait.Umbraco.Search.Typesense/Lait.Umbraco.Search.Typesense.csproj
```

Backoffice client (produces `App_Plugins/.../typesense.js`, which the `.csproj` packs into the NuGet package):

```bash
cd src/Lait.Umbraco.Search.Typesense/Client
npm install
npm run build
```

See [`src/Lait.Umbraco.Search.Typesense/readme.md`](src/Lait.Umbraco.Search.Typesense/readme.md)
for configuration and usage.
